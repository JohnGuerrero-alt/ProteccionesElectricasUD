Pasos

1. Para comenzar la coordinación de los Relés, se comienza desde el relé C hasta el A. Es importante ingresar las variables Isc, Ic, Incremento Ic, Familia de curvas y de su elección (el tiempo o TMS). Para modificar los valores de cada relé, presione la imagen del relé que quiere configurar .

2. Presione el botón "Calcular". En la pestaña "Coordinación", el programa realizará el proceso de ajuste de coordinación empezando con el relé C, continuando con el relé B y por último el relé A.

Nota: Si se presenta alguna novedad, en la barra que contiene el texto "Estado", avisará si es necesario cambiar el valor de alguna variable o se completo el proceso exitosamente.

3. Una vez finalice el programa de realizar los ajustes correspondientes, habilitará la pestaña "Grafica", donde se visualizará la curva de tiempo inverso según los datos suministrados.

Estado
Los datos iniciales de cada uno de los relés han sido cargados. La coordinación comienza desde el relé C. Hay dos formas para realizar el proceso de coordinación:

1. Variable "TIEMPO"
Si desea que el programa realice la coordinación a partir del tiempo de operación (Calcula el valor del TMS). Asigné el valor del tiempo en cada uno de los relés.

2. Variable "TMS"
En cambio si desea que el programa realice la coordinación a partir del DIAL (Calcula el tiempo de operación). Asigne el valor del DIAL para el relé C y si lo desea también para el relé B y relé A.

Observación: Si asigna valores en ambos campos, el programa tomará como prioridad el valor del DIAL.

Pasos

1. Para comenzar asigne los valores de la potencia, voltaje y el valor Z de cada uno de los elementos del anillo, dando click en la imagen del elemento a configurar .

3. Seleccione los valores faltantes para al final poder graficarlo.
