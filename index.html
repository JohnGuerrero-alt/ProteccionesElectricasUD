Pasos

1. Para comenzar la coordinación de los Relés, se comienza desde el relé C hasta el A. Es importante ingresar las variables Isc, Ic, Incremento Ic, Familia de curvas. Para modificar los valores de cada relé, presione la imagen del relé que quiere configurar .

2. Presione el botón 'Calcular' ubicado en la pestaña "Coordinación", el programa requerirá que escoja la variable con la cual trabajará para realizar el proceso de ajuste de coordinación (Tiempo de operación o TMS), administre los datos correspondientes a la variable con la que necesita trabajar.

3. Una vez administrado los datos y seleccionado la variable a trabajar, vuelva a presionar el botón 'Calcular'. El programa realizá el proceso de coordinación empezando con el relé C, continuando con el relé B y por último el relé A.

Nota: Si se presenta alguna novedad, en la barra que contiene el texto "Estado", avisará si es necesario cambiar el valor de alguna variable o si completo el proceso exitosamente.

4. Una vez finalice el programa de realizar los ajustes correspondientes, habilitará la pestaña "Grafica", donde se visualizará la curva de tiempo inverso según los datos suministrados.
